## 📦 What's Included

This pack automatically installs and configures the following tools:

- **Tailwind CSS IntelliSense** (`bradlc.vscode-tailwindcss`) - Intelligent Tailwind CSS tooling.
- **Bearded Theme** (`BeardedBear.beardedtheme`) - A beautiful, eye-friendly theme collection.
- **Flow Icons** (`thang-nm.flow-icons`) - Clean and modern file iconography.
- **Prettier** (`esbenp.prettier-vscode`) - Opinionated, consistent code formatter.
- **Docker** (`ms-azuretools.vscode-docker`) - Seamless container management directly from your editor.

## 🚀 How to Use

1. Once installed, the extensions will download in the background.
2. Press `Ctrl+Shift+P` (or `Cmd+Shift+P` on macOS) to select the **Bearded Theme** and **Flow Icons** if they don't apply automatically.
3. Enjoy a secure, unified development environment!
